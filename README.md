# School-Management-System
A full-stack web application for high school administration. Features student/teacher management, grade tracking, and admin dashboard. Built with ASP.NET MVC 5, C#, and SQL Server.
# 🏫 High School Administration System

A full-stack web application designed to digitize and manage high school records, student grades, and teacher assignments. This project demonstrates **CRUD operations**, **Role-based Authentication**, and **Relational Database Management**.

## 🚀 Key Features
- **Admin Dashboard:** Statistical overview of students, teachers, and courses.
- **User Management:** Secure login and administrative control over user roles.
- **Student Records:** Comprehensive tracking of student profiles and academic history.
- **Gradebook System:** Real-time entry and management of student grades by subject.
- **Data Persistence:** Automated database interactions using Entity Framework.

## 🛠 Tech Stack
- **Backend:** C# (ASP.NET MVC 5)
- **Database:** SQL Server (Entity Framework)
- **Frontend:** Bootstrap, jQuery, HTML5, CSS3
- **Architecture:** MVC (Model-View-Controller)

## 📋 How to Run
1. **Clone the repository** to your local machine.
2. **Execute the `database_setup.sql` script** (found in the root folder) in your SQL Server Management Studio.
3. **Update the `Web.config` file** with your local SQL Server instance name.
4. **Open the solution** (`.sln` file) in Visual Studio and press **F5**.
